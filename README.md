### 1. Clone the Repository:
```sh

cd microproject-localcoffee
```

### 2. Install Dependencies:
```sh
npm install
```

### 3. Run the Server with Nodemon:
```sh
npm start
```

### 4. Open in Browser:
[http://localhost:3000](http://localhost:3000)
